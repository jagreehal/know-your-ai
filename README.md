# Know Your AI

A fun game to test your knowledge of AI company logos. Match AI company icons to their names!

## 🚀 Live Game

Play the game at: https://jagreehal.github.io/know-your-ai

## 🏗️ Tech Stack

- Astro + React
- TailwindCSS
- Framer Motion
- shadcn/ui

## 🚀 Project Structure

```
/
├── public/
├── src/
│   ├── components/
│   ├── pages/
│   │   └── index.astro
│   └── styles/
└── package.json
```

## 🧞 Commands

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
